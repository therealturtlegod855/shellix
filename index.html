Shellix OS Overview

Shellix OS is used much like Ubuntu, with similar workflows and tools but some different configuration choices.

How to use

Use this the same as Ubuntu; it is nearly the same except for some configs.

  • Base experience: ubuntu like with a kde enviornment and preinstalled dev tools to stramline dev workflow
  • Desktop: Kde plasma
  • Config differences: Desktop, Branding, Packages
